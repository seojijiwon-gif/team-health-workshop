# 배포 가이드 — 처음 하시는 분을 위한 단계별 설명

---

## 준비물
- 컴퓨터 (윈도우 / 맥 모두 가능)
- 이메일 계정 1개

---

## 파일 구조 확인

배포 전에 폴더 구조가 아래와 같은지 확인하세요.

```
team-health-workshop/       ← 폴더 이름 (자유롭게 변경 가능)
├── public/
│   └── index.html          ← 앱 화면 전체
├── api/
│   └── feedback.js         ← AI 피드백 처리
├── vercel.json             ← 배포 설정
└── package.json
```

---

## STEP 1 — GitHub 가입 & 레포지토리 만들기

GitHub는 코드를 저장하는 공간입니다. 구글 드라이브와 비슷하다고 생각하면 됩니다.

1. 브라우저에서 **github.com** 접속
2. 우측 상단 **Sign up** 클릭
3. 이메일, 비밀번호, 사용자명 입력 후 가입 완료
4. 가입 후 로그인된 상태에서 우측 상단 **+** 버튼 클릭 → **New repository** 클릭
5. 아래와 같이 입력:
   - Repository name: `team-health-workshop`
   - 나머지는 기본값 유지
6. 초록색 **Create repository** 버튼 클릭

---

## STEP 2 — 파일 업로드

레포지토리가 만들어지면 파일을 올려야 합니다.

1. 생성된 레포지토리 페이지에서 **uploading an existing file** 링크 클릭
   (또는 **Add file** → **Upload files** 클릭)
2. 탐색기(파인더)에서 `team-health-workshop` 폴더 안의 **모든 파일과 폴더**를 선택
3. 브라우저 업로드 창에 드래그 앤 드롭
4. 페이지 하단 **Commit changes** 버튼 클릭

> ⚠️ 주의: 폴더째로 드래그하면 안 됩니다.
> `public` 폴더, `api` 폴더, `vercel.json`, `package.json` 파일을 각각 선택해서 올리세요.

---

## STEP 3 — Anthropic API 키 발급

AI 피드백 기능을 사용하려면 API 키가 필요합니다.

1. **console.anthropic.com** 접속
2. **Sign up** 으로 회원가입 (이메일 인증 필요)
3. 로그인 후 왼쪽 메뉴에서 **API Keys** 클릭
4. **Create Key** 버튼 클릭 → 키 이름 아무거나 입력 → **Create Key**
5. 화면에 표시되는 `sk-ant-api03-...` 로 시작하는 값을 **복사해서 메모장에 저장**
   (이 창을 닫으면 다시 볼 수 없습니다!)

> 최초 가입 시 무료 크레딧이 제공됩니다.
> 팀 워크숍 수십 회를 진행할 수 있는 분량입니다.

---

## STEP 4 — Vercel 가입 & 배포

Vercel은 GitHub에 올린 코드를 인터넷 주소로 만들어주는 서비스입니다.

1. **vercel.com** 접속
2. **Start Deploying** 또는 **Sign Up** 클릭
3. **Continue with GitHub** 선택 → GitHub 계정으로 로그인
4. 로그인 후 **Add New Project** 클릭
5. GitHub 레포지토리 목록에서 `team-health-workshop` 옆 **Import** 클릭
6. 설정 화면이 나오면 아무것도 바꾸지 말고 파란색 **Deploy** 버튼 클릭
7. 1~2분 후 배포 완료! 🎉
   - 화면에 `team-health-workshop.vercel.app` 형태의 주소가 생성됩니다.

---

## STEP 5 — API 키 등록

지금은 AI 피드백이 동작하지 않습니다. API 키를 등록해야 합니다.

1. Vercel 대시보드에서 방금 배포한 프로젝트 클릭
2. 상단 탭에서 **Settings** 클릭
3. 왼쪽 메뉴에서 **Environment Variables** 클릭
4. 아래와 같이 입력:
   - **Name** 칸에: `ANTHROPIC_API_KEY`
   - **Value** 칸에: STEP 3에서 복사한 `sk-ant-...` 값 붙여넣기
5. **Save** 버튼 클릭
6. 상단 탭 **Deployments** 클릭
7. 목록 맨 위 항목 오른쪽 **...** 클릭 → **Redeploy** 클릭
8. 팝업에서 **Redeploy** 클릭

---

## 완료!

Vercel에서 생성된 주소(예: `team-health-workshop.vercel.app`)를
팀원들에게 링크로 공유하면 바로 사용할 수 있습니다.

## 자주 묻는 질문

**Q. 비용이 얼마나 드나요?**
GitHub 무료 / Vercel 무료 / Anthropic API는 팀 1회 워크숍당 약 10~50원 수준입니다.

**Q. 코드를 수정하고 싶으면 어떻게 하나요?**
GitHub에서 파일을 직접 수정하면 Vercel이 자동으로 재배포합니다.

**Q. 주소를 바꾸고 싶어요.**
Vercel 대시보드 → Settings → Domains에서 원하는 이름으로 변경 가능합니다.
